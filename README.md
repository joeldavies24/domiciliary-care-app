# Domiciliary Care Web Application

A web application desgined for domiciliary care services designed to manage Carer visitations to service users homes.

## Description

An application designed to create and improve upon standardised domicilary care systems based on feedback provided by care workers. This web application was created for my final year project and was created using Node.js, Express, Passport, MongoDB, EJS, and CSS. 

## Features

- User authentication 
- Role-based access
- Manage service users
- Schedule and track care visits
- Secure session handling with Passport.js

## Getting Started

### Dependencies

* Node.js
* npm
* MongoDB
* Any modern web browser

### Installing

1. Clone the repository:
   
git clone https://github.com/joeldavies24/domiciliary-care-app.git

2. Navigate into the project folder:

cd domiciliary-care-app

3. Install required dependencies:

npm install

4. Configure the database

In dbconfig.js, change the string value `const mongoDB = "";` to match your MongoDB server

---

### Executing program

1. Start the server:

node app.js

2. Open your browser and go to:

http://localhost:3000
